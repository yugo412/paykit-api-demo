# Paykit API Demo

A simple API for demonstrating integration with [paykit-php](https://github.com/yugo412/paykit-php).

## Base URL

```
https://paykit-api-demo.fly.dev
```

## Endpoints

### Get All Products

```
curl --request GET \
  --url https://paykit-api-demo.fly.dev/product
```

### Get Product Detail

```
curl --request GET \
  --url https://paykit-api-demo.fly.dev/product/1
```
